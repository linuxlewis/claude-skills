# Agent Browser Plugin

Browser automation skill for Claude Code using the `agent-browser` CLI.

## Features

- Navigate websites
- Interact with page elements (click, type, fill forms)
- Take screenshots and PDFs
- Extract content from pages
- Handle forms and interactive elements

## Installation

```bash
claude plugin install agent-browser@linuxlewis-skills
```

## Requirements

The `agent-browser` CLI must be installed:

```bash
npm install -g agent-browser
agent-browser install
```

## Usage

Once installed, Claude will automatically use this skill when you ask to:
- Browse a website
- Fill out forms
- Click buttons
- Take screenshots
- Scrape content
